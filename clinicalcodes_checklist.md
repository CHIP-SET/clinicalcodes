Checklist for clinical codes research objects
=============================================


Level 1: Article
-----------------

### MUST

* URL
* Title
* Journal
* Authors

### SHOULD

* Abstract
* DOI

### COULD

* Comments
* Article type 

Level 2: Codelist
-------------------

### MUST

* Name
* associated articles
* URL

### SHOULD


### COULD

* Comments

Level 3: Clinical codes
-----------------------

### MUST

* code name
* coding system
* codelist name

### SHOULD

* Description
* Entity type
* Number of codes

### COULD

* User specified fields
